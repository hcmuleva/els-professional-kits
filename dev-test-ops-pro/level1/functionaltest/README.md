## 🚀 Quick Start

### Prerequisites
- Java 11 or higher
- Maven 3.6+
- Strapi API running at: http://localhost:1337/api

### Installation
```bash
# Install dependencies
mvn clean install -DskipTests
```

### Run Tests
```bash
# Run all tests
mvn test

# Run Level 1 tests only
mvn test -Dtest=TestRunner#testLevel1

```
